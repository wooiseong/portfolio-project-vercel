  • "與coding相遇的契機"

    Apa khabar! 我是一名民國85生的馬來西亞人，畢業於成功大學微生物與免疫研究所。

    在畢業後，我擔任長庚大學的研究助理，主要負責案子的組織染色，顯微鏡拍攝和圖像處理。

    實驗性質的工作讓我學會自己查找英文期刊和訓練邏輯能力———實驗影響染色結果的機制。把案子呈現給老闆時，應對老闆的提問，培養我有效溝通和隨機應變的能力。

    使用統計分析軟體時，我對於軟體界面如何與人互動產生很大興趣。在業餘自主學習javascript的知識時，發現coding能帶給我滿足感，特別是解決了bug的時候。

    因此，我計劃朝著前端工程之路邁進。

  • "積極學習，全神貫注"

    我的學習路徑是 HTML > CSS > JS > AJAX和API > Vue。

    在6個月學習的過程中，我一面觀看綫上課程，一面敲代碼。我做了幾個小小的項目驗證我的學習。

    雖然遇到很多技術的問題，但過去研究所查詢文獻，鎖定問題和過濾信息的能力讓我很快理清問題。也感謝IT邦，medium，stack overflow和github的code友們的慷慨幫助。

    在準備作品時，其中最大的挑戰是：

    使用者點擊注冊后，下一頁應該要切換成注冊，而不是默認的登入頁面。我利用路由跳轉傳參數的方式，讓下一頁根據參數切換成對應頁面。

    審視現在的自己，其實很喜歡coding的生活，決定換個跑道，加入coding的世界。

  • "提升效率，成為專業"

    進入網絡工程的行業后，我期許是一位注重網頁流暢度，用戶使用體驗和開發效率的前端工程師。

    我明白自學的知識和業界有段距離。在工作時，希望可以瞭解企業在開發網頁的流程，鞏固自己的知識，培養企業框架開發思維。

    未來計劃有三：
    （一）把過去的作品代碼做unit test，刪除冗餘代碼，優化代碼結構。

    （二）整理所學習和業界接觸的知識，以部落格形式整理，理解所學知識點，提升開發效率。

    （三）深入接觸資料結構和演算法相關的知識，提升對程式語言的掌握度。
